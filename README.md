# Cardano Savings Vault DApp

A decentralized piggy bank for ADA, built with Haskell (Plutus logic) and Javascript.

## Features
- **Smart Staking**: Lock ADA until a target is met.
- **Haskell Validator**: Vault logic prevents withdrawal unless target >= balance.
- **Vercel Ready**: Pre-architected for instant deployment.

## Deployment
1. Connect Nami/Eternl wallet.
2. Set a savings goal in ADA.
3. Deposit as you go.
4. Notification appears & Withdrawal unlocks automatically when goal is hit.
